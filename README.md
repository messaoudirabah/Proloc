# ProLoc | High-Performance Car Rental Management 🚗💨

ProLoc is an elite Car Rental Management ecosystem designed for high-precision fleet control. Built with Python and PyQt5, it features a cinematic HUD for tracking vehicles, clients, and active rentals with integrated PDF contract generation. It includes advanced financial analytics, automated refueling logs, and multi-theme support for a next-level agency operation.

## 🚀 Key Features

*   **Fleet Mastery**: Complete control over vehicle brands, models, and real-time status (Available, Rented, Repair).
*   **Client Ecosystem**: Secure management of personal data, license scans (Recto/Verso), and digital contracts.
*   **Rental Intelligence**: Automated rental calculation, insurance policy tracking, and deposit management.
*   **Advanced Analytics**: Real-time financial reports, revenue-vs-expense charts, and refueling cost logs.
*   **High-Fidelity UI**: Cinematic PyQt5 interface with smooth animations and multi-theme support (Dark/Light).
*   **PDF Generation**: Dynamic generation of contracts, factures, and records in both French and Arabic.

## 🛠️ Technical Stack

*   **Core Logic**: Python 3.x
*   **Interface**: PyQt5 (Dynamic HUD)
*   **Database**: SQLite3 (Optimized)
*   **Reporting**: ReportLab (Bi-lingual PDF Support)
*   **Packaging**: PyInstaller & Inno Setup

## 📦 Quick Start

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/proloc.git
    cd proloc
    ```

2.  **Install Essential Core**:
    ```bash
    pip install PyQt5 reportlab arabic-reshaper python-bidi matplotlib
    ```

3.  **Execute Phase**:
    ```bash
    python main.py
    ```
